## :computer: Kurulum ve Kullanım

1. Projeyi klonlayın:
```bash
git clone https://github.com/tunahanyasar/Hafta-8-Sport-Center.git
```
2. Proje dizinine gidin:
```bash
cd Hafta-8-Sport-Center
```
3. `index.html` dosyasını bir web tarayıcısında açın.
